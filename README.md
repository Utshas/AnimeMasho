# AnimeMasho
A practice project to learn android animation. Let's Animate.

This is a kotlin project, where I have coded to animate a textview in 12 different ways. They are:
1.Slide-up
2.Slide-down
3.Slide-left
4.Slide-Right
5.Rotate
6.Move
7.Bounce
8.Zoom-in
9.Zoom-out
10.Fade_in
11.Fade_out
12.Blink
